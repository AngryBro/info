Психологические исследования

Платформа, реализующая создание опросов для проведения психологических исследований. Включена обработка данных в виде разделения опрошенных на группы. Результаты опросов представляются в виде таблицы с подсчитанными баллами по шкалам. Таблицы можно скачать в формате Excel. Данные можно статистически обработать по критерию Фишера.

Проект представлялся в качестве выпускной квалификационной работы по направлению «Математика и компьютерные науки» в ДВФУ.

Класс для работы с матрицами

Функциональный класс на языке Javascript для работы с матрицами. Поддерживает стандартные операции, такие как: сложение, вычитание, умножение, обращение матрицы, вычисление определителя, вычисление миноров и алгебраических дополнений. Позволяет работать с матрицей как с вектором. Поддерживается вычисление матричных и векторных норм. Различные форматы выводы матрицы: в консоль, преобразование в массив, в код Latex, в строку.

Нейронная сеть

Проект разрабатывался в личных целях для изучения принципа работы нейронных сетей. В нём реализован класс для создания и обучения нейронной сети, решающей задачу регрессии. Тестирование было проведено на iris dataset. Точность бинарной классификации достигала 100%, а тернарной – 95%.
